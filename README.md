## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```
* Optional: yarn
  ```sh
  npm install --global yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/iremlaya/github-search.git
   ```
2. Install NPM packages. I prefer Yarn.
   ```sh
   yarn install
   ```
3. Start the project.
    ```sh
   yarn start
   ```
